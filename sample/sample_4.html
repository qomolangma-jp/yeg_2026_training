50th Anniversary Architecture

伝統と地域経済が
交差する二日間。どんどん祭り 2025

小松市で約50年に渡り紡がれてきた、街全体の鼓動を高める商業的な祭事。洗練された伝統のインフラと現代のエンターテインメントが高度に融合します。

コア・プログラム

洗練された伝統工芸の展示から、熱狂を創出するライブステージまで包括的なイベントレイアウト。

EVENT // 01

輪島キリコ展示

能登のアイデンティティである巨大な伝統造形「キリコ」が小松へ。漆黒に映える壮大な光のインスタレーションを体感。

EVENT // 02

行燈行列

柔らかな陰影を描く行燈（あんどん）の光波が、夜の街を静かに美しく、幻想的なスケールで彩ります。

EVENT // 03

DJコーによるライブ

特別ゲスト「DJコー」がメインステージに登壇。圧倒的な音響アプローチで、会場全体に圧倒的なグルーヴを共有します。

EVENT // 04

ダンスイベント

選び抜かれたパフォーマーたちが躍動する、高エネルギーなダンスステージ。熱量と洗練が交差する瞬間。

EVENT // 05

キッチンカー ＆ 縁日

現代的な美食アプローチを提供するキッチンカー群と、伝統的な祭事の楽しさをリビルドしたハイクオリティな縁日。

EVENT // 06

ステージパフォーマンス

多種多様なアーティストや団体が、二日間にわたりノンストップで最高峰のエンターテインメントを提供します。

開催概要

イベント名称 どんどん祭り
タイムフレーム 2025年10月11日（土） 〜 10月12日（日）
ロケーション 小松駅前 特設会場
プラットフォームの特徴 小松市にて約50年の実績を持つ、地域経済の活性化と文化的調和を目的とした最大級の商業祭。
